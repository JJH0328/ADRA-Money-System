# ADRA-Money-System

## 무슨 애드온 인가요?
본 애드온은 게리모드 ADRA(시리즈)에 속해 있는 하나의 애드온입니다.    
포인트샵으로 돈 시스템을 대신하고 있는 서버를 주 타겟층으로 온전한 돈 시스템을 사용할 수 있게 개발되었습니다.    
자신만의 서버에 사용하기 위해 제작된 애드온을 고려하여 API를 지원합니다. 자세한 내용은 하단부분에서 참고바랍니다.
